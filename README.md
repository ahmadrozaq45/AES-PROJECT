## How to Run

### 1. Clone Repository
```bash
git clone https://github.com/ahmadrozaq45/AES-PROJECT
cd AES-PROJECT
```
### 2. Install Requirements
```bash
pip install -r requirements.txt
```
### 3. Run Project
```bash
streamlit run app.py
```
if it doesn't work, use this
```bash
python -m streamlit run app.py
```
